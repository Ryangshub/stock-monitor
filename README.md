# 주식 데이터 자동 수집기

국내 주식(pykrx) / 해외 주식(yfinance)을 매시간 자동 수집하여 엑셀에 기록합니다.

---

## 파일 구성

```
stock_monitor/
├── stock_collector.py   ← 메인 프로그램
├── requirements.txt     ← 패키지 목록
├── 관심종목.xlsx         ← 설정 + 관심종목 + 수집 데이터 (자동 생성)
└── stock_collector.log  ← 실행 로그 (자동 생성)
```

---

## 설치

```bash
pip install -r requirements.txt
```

---

## 엑셀 파일 구조

| 시트명 | 역할 |
|--------|------|
| `설정` | 시작시간, 종료시간, 수집간격(분) 설정 |
| `관심종목` | 수집할 종목 리스트 |
| `종목코드_종목명` | 종목별 수집 데이터 (자동 추가) |

### 관심종목 시트 입력 방법

| 종목코드 | 종목명 |
|---------|--------|
| 005930  | 삼성전자 |
| 000660  | SK하이닉스 |
| AAPL    | Apple Inc. |

- **국내 주식** : 6자리 숫자 코드 (KRX 코드)
- **해외 주식** : 영문 티커 (예: AAPL, MSFT, NVDA, TSLA)
- 종목명은 비워도 되며, API에서 자동으로 가져옵니다.

### 설정 시트

| 항목 | 기본값 | 설명 |
|------|--------|------|
| 시작시간 | 09:00 | 수집 시작 시각 (HH:MM) |
| 종료시간 | 16:00 | 수집 종료 시각 (HH:MM) |
| 수집간격(분) | 60 | 60 / 30 / 15 중 선택 (60의 약수) |

## 수집 데이터 항목

| 열 | 설명 |
|----|------|
| 수집시각 | 수집 실행 시각 |
| 종목코드 | KRX 코드 또는 티커 |
| 종목명 | 종목 이름 |
| 현재가 | 최신 종가 (장외·주말은 직전 거래일 종가) |
| 거래량 | 해당 일 거래량 |
| 등락률(%) | 전일 종가 대비 등락률 |
| 한달최고가 | 최근 30일 고가 최대값 |
| 분기최고가 | 최근 90일 고가 최대값 |
| 반기최고가 | 최근 180일 고가 최대값 |

## 실행 방법

### 1) 초기 템플릿 생성
```bash
python stock_collector.py --init
```
`관심종목.xlsx` 파일을 생성합니다. 관심종목과 설정을 입력한 뒤 아래 명령으로 실행하세요.

### 2) 스케줄러 모드 (자동 반복 수집)
```bash
python stock_collector.py
# 또는 경로 직접 지정
python stock_collector.py "D:\stock\watchlist.xlsx"
```
설정 시트의 시작/종료 시간, 수집 간격에 따라 자동으로 반복 수집합니다.  
종료: `Ctrl+C`

### 3) 1회 즉시 수집
```bash
python stock_collector.py --once
```

## Windows 작업 스케줄러 연동 (선택)

PC 재시작 후에도 자동 실행하려면 Windows 작업 스케줄러에 등록하세요.

1. **작업 스케줄러** 열기 (검색: `taskschd.msc`)
2. **작업 만들기** → 이름: `주식수집기`
3. **트리거**: 로그온 시 / 또는 매일 시작시간에 시작
4. **동작**: 프로그램 시작
   - 프로그램: `python`  또는 `C:\경로\python.exe`
   - 인수: `"C:\Users\subht\Documents\stock_monitor\stock_collector.py"`
5. **조건**: "AC 전원 공급 시에만 실행" 체크 해제

> 스크립트 자체에 APScheduler가 내장되어 있으므로,  
> 작업 스케줄러는 **프로그램 시작** 역할만 담당하면 됩니다.

## 주의사항

- 엑셀 파일이 열려 있으면 저장에 실패합니다. 수집 중에는 파일을 닫아두세요.
- pykrx는 장 마감 후(~16:00 이후) 데이터가 안정적으로 제공됩니다.
- 해외 주식 수집 시 네트워크 연결이 필요합니다.
